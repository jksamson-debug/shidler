5 Accessibility

Ensure accessible contrast ratios for all text and interactive elements.

  • Primary text (#1A1A1A) on beige (#FAF7F0) — contrast ratio ~15:1, exceeds WCAG AAA
  • Accent colors (yellow, blue, green, purple) must not be used for text on light backgrounds — insufficient contrast
  • Red (#EF5331) is acceptable for large text (18pt+) and UI elements but not body copy
  • Minimum touch target: 44px for all interactive elements
  • Support prefers-reduced-motion for marquee and transition animations
