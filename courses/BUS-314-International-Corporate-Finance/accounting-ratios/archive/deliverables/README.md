# Deliverables

Store your final outputs here — polished memos, completed spreadsheets, and presentation materials.

## Typical Contents

- `stage1-memo-LASTNAME.md` — Executive memo (Stage 1)
- `stage2-model-LASTNAME.xlsx` — Excel model (Stage 2)
- `stage3-spec-LASTNAME.md` — Technical specification (Stage 3)
- `stage4-analysis-LASTNAME.md` — Final analysis, prompt & recommendation (Stage 4)
- `extra-credit-LASTNAME.md` — Extra credit addendum (if applicable)

## Quality Checklist

Before submitting each deliverable:
- [ ] File follows the naming convention
- [ ] Content meets all assignment requirements
- [ ] Professional formatting and tone
- [ ] Sources and assumptions documented
- [ ] AI usage documented (if applicable)
