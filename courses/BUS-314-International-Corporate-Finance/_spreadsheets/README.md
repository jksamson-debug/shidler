# BUS-314 Master Spreadsheets

This directory contains the master reference spreadsheet for the Accounting & Performance Ratios project.

## Files

| File | Description |
|------|-------------|
| `BUS-314 Accounting & Performance Ratios - MASTER.xlsx` | Instructor reference model with all 25+ ratios across 6 categories |

## Usage

This spreadsheet serves as the authoritative reference for:
- Named range conventions (`BAL_`, `INC_`, `CASH_`, `RATIO_` prefixes)
- Ratio formula definitions
- Color coding standards (Yellow = inputs, Blue = assumptions, Green = formulas, Gray = outputs)
- Expected output format

Students should use their assigned scenario workbook in `accounting-ratios/_scenarios/` as the starting point for Stage 2, not this master file.
